# Airbnb Price Classification using Machine Learning

## Overview

This project was completed as part of the **Break Through Tech AI Machine Learning Foundations** curriculum (Modules 1–5), a nationally recognized AI program in partnership with Cornell Tech.

The project applies the concepts learned throughout the curriculum, including Python programming, data preprocessing, exploratory data analysis (EDA), feature engineering, supervised machine learning, model selection, hyperparameter tuning, and model evaluation to classify Airbnb listings into price categories.

---

## Project Objectives

- Explore and clean a real-world Airbnb dataset
- Perform exploratory data analysis (EDA)
- Engineer and select predictive features
- Build a Logistic Regression classification model
- Optimize model performance using GridSearchCV
- Evaluate model performance using industry-standard classification metrics
- Serialize the trained model for future deployment

---

## Dataset

- Airbnb listings dataset
- **28,000+ Airbnb listings**
- Target Variable: **Price Category (High vs. Low)**
